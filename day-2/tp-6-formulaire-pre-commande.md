# TP formulaire de pré-commande

- Créer un formulaire de pré-commande, ajouté sur les pages produits

- L’utilisateur renseigne la quantité, son nom et prénom, son email

- Valider les valeurs des champs avec zod

- **Bonus** : ajouter les données dans une base de données grâce à Prisma

- **Bonus** : permettre d'**aimer un produit** depuis la liste de produits sur la page d'accueil. Au clic sur le like, appel d'une **route API** qui **enregistre le like** dans la base de données. L'**ajout du cookies et la lecture** de celui-ci permettent de savoir que l'utilisateur a déjà aimé le produit.
