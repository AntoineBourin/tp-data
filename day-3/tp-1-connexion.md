# TP : remplacez votre système de connexion

- Créez une action serveur à la place de votre API route

- Création, écriture et lecture de vos cookies directement dans l’action

- Retourner un message d’erreur / succès à afficher dans l’interface

- **Bonus** : ajoutez une action serveur à votre formulaire de pré-commande, ajoutez la pré-commande en base de données. Lors de l’ajout, revalidez les données de la page qui liste les pré-commandes. Vous pouvez rediriger vers cette page.
