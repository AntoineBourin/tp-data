# TP : implémenter une inscription/connexion d’un utilisateur

- Créer un formulaire d’inscription
  Lors de la soumission du formulaire, ajouter l’utilisateur à la base de données

- Créer un formulaire de connexion
  Lors de la soumission du formulaire, vérifier que l’utilisateur soit présent dans la base de données et le connecter.

- **Bonus** : ajouter un bouton « J’aime » sur la liste des produits, ajouter les produits préférés dans une liste sur le profil utilisateur
